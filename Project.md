Our project is based on sales data analysis of Vrinda Store. We worked on a retail dataset containing order, customer, and product details. Also performed data cleaning and analysis in Excel and created an interactive dashboard using charts and slicers. The project helped in identifying sales trends,top categories, and regional performance. Through this, we developed strong skills in Excel, data visualization, and analytical thinking.
